FLR-1000 型自动发酵系统安装使用说明书

一、前言
1. 设备基础信息
  • 型号：FLR-1000
  • 核心布局：罐体（左）+ 柜体（右），间距53cm，罐体投料口（红色盖子）朝前，顶部横梁与柜体水平居中对齐
  • 管路连接要求：丝口连接处需缠生料带（活接除外），直插接口需涂抹704胶水！
设备整体示意图

2摆放罐体与柜体
操作说明：按“罐体在左、柜体在右”的顺序摆放，间距控制在53cm左右；确保罐体的投料口（红色盖子）朝前，罐体顶部横梁与柜体水平居中对齐。如需调整布局，需根据场地尺寸另配管件。
管路系统示意图
3安装罐内循环管路
连接顺序：罐顶循环口内 → 活接 → 循环直接管 → 循环弯接管
弯头出水口需要对准筒壁
4安装底部出料管路
连接顺序：罐体最下端出料口 → 大活接 → 循环直接管 → 弯头 → 直接管（4） → 三通 → 直接管（6） → 手动阀 → 直接管（4） → 活接 → 柜体出料口
手动阀应略微向下倾斜，便于排污干净
5安装液位显示管路
连接顺序：罐体下端液位接口 → 三通（20，垂直向上） → 垂直透明管 → 罐体上端液位接口 → 连接管（8） → 三通（20，垂直向下连接透明管） → 向上手动阀 → 连接管（9）
垂直透明管需保持竖直安装，确保液位显示准确；接口密封处理同前
6安装清洗管路
连接顺序：罐体顶端中心清洗接口 → 活接 → 连接管（7） → 弯头 → 连接管（2） → 三通（32转25） → 向下连接液位顶端连接管（9） → 向右连接管（5） → 弯头 → 连接管（1） → 弯头 → 直接 → 活接 → 柜体清洗口
确保罐体顶端清洗球头居中安装，保证后续清洗覆盖全面
7安装进水管路
连接顺序：罐体顶端进水口 → 活接 → 直接 → 弯头 → 连接管（3） → 弯头 → 连接管（1） → 弯头 → 直接 → 活接 → 柜体进水口
管路走向需避开其他管路，避免交叉缠绕
8安装循环管路
连接顺序：罐体顶端循环口 → 活接 → 直接 → 弯头 → 连接管（3） → 弯头 → 连接管（1） → 弯头 → 直接 → 活接 → 柜体循环口
管路走向需避开其他管路，避免交叉缠绕
9安装温度感应器
操作说明：
  1. 将温度探针从液位显示管的下端三通处插入并拧紧；
  2. 将电线插接头插入柜体航空插座（三针），拧紧盖帽；
  3. 用扎带将电线与管道捆扎固定，避免拉扯。
10安装液位感应器
操作说明：
  1. 将连接线较长的探头卡在透明管的上端，连接线较短的探头卡在透明管最下端；
  2. 将插头插入柜体航空插座（四针），拧紧盖帽；
  3. 用扎带将电线与管道捆扎固定，确保探头安装牢固。
11连接进水和出料管道
操作说明：将自来水进水管道连接到柜体右侧进水口，将出料管道连接到柜体右侧出料口，建议均采用活接连接，便于后续维护拆卸。
12连接电源线
操作说明：将电源线从柜体左侧预留孔穿入柜体，与柜体内的电源接线端子进行连接。建议由专业电工操作，确保用电安全。
电源线连接需专业电工操作，避免触电风险
13运行调试
控制柜面板示意图
1) 测试通电
操作：按“电源开关”→“运行”，观察PH显示器、温度显示是否均点亮，点亮则说明通电正常。
2) 进水测试
操作：
  1. 按“进水控制”开关，进水阀门自动开启向罐体注水，观察所有管路是否松动漏水，如有漏水需及时紧固；
  2. 将高液位探头移动到300L刻度线左右，低液位探头移动到最下端；
  3. 观察水位到达低液位时，低液位探头红灯是否点亮；水位到达高液位时，高液位探头红灯点亮，且进水阀门自动关闭。
3) 升温循环测试
操作：
  1. 按“温度控制”开关，同时将档位开关调整到低档，观察水泵是否启动，罐体内是否开始循环搅拌；
  2. 几分钟后观察温度是否开始升高，同时检查所有管路是否渗漏，如有漏水需紧固；
  3. 关闭“温度控制”开关，档位开关调至“关”，单独按“循环控制”开关，观察罐体内部的水是否正常搅拌。
4) 清洗出料测试
操作：
  1. 按“清洗控制”开关，观察罐体顶端不锈钢清洗球头是否正常喷出高压水柱，实现全方位清洗；
  2. 关闭“清洗控制”开关，按“出料控制”开关，罐体内的水经出料管路排出，直至排空后水泵自动停止运行。
5) PH值校准
说明：出厂时已完成校准，如需进一步校准，可参考《PH校准操作说明书》。
6) 温度设定
说明：出厂时已预设温度范围，如需调整，可参考《温度设定操作说明书》。
三、使用教程
1. 发酵前准备
操作：对于新设备或中断发酵一段时间的设备，需进行2-3次彻底清洗：
  1. 按“清洗控制”开关，系统自动进水到低液位并启动水泵冲洗，每轮约2分钟，“清洗控制”开关绿灯闪烁，水泵自动停止，此时须手动打开底部排污阀进行排污，直至全部排尽后再手动关闭“清洗控制”开关；
  2. 如果“清洗控制”开关绿灯闪烁时，需要快速通过出料管道排水，必须先手动关闭“清洗控制”开关，再按“出料控制”开关通过出料管路快速排水；
  3. 重复上述程序2-3次，确保设备清洁无杂菌。
2. 开始发酵
操作：
  1. 调整高液位探头到设定发酵液位刻度（不得低于最低液位）；
  2. 按“进水控制”开关，系统自动上水，到达设定液位后自动关闭进水阀门；
  3. 从投料口按比例投入发酵剂（大华农益菌王0.5%），关闭投料口桶盖；
  4. 将档位开关调整到对应档位（发酵液大于500L调至“高”，小于500L调至“低”）；
  5. 按“温度控制”开关，发酵正式启动，进入无人值守状态。
3. 终止发酵
操作：当PH值低于3.8时，可随时终止发酵：关闭“温度控制”开关，将档位开关调整到“关”，发酵终止。
4. 出料
操作：
  1. 出料前先按“循环控制”开关，运行1分钟左右，让发酵液充分混合均匀；
  2. 按“出料控制”开关，将发酵液转移到周转桶；
  3. 出料完毕后，系统自动关闭水泵。
5. 日常清洗维护
  • 每天连续发酵：建议每周清洗1次；
  • 中断发酵一段时间：最后一次发酵出料结束与下次重新使用前需清洗2-3遍；
  • 清洗后需拧紧桶盖，防止滋生杂菌。
四、注意事项
发酵剂使用限制：本产品仅针对大华农益菌王（水溶型）发酵剂开发，不适用其他任何含有非水溶成分的发酵剂产品（如添加饲料等），也不适用于其他需要特殊环境控制的菌种发酵！
使用环境要求：本产品未经防水抗冻设计，必须在室内环境使用，不得在户外无任何防护措施的环境中使用！
消毒注意事项：可对罐体及控制柜表面进行喷洒消毒，切勿打开控制柜柜门对内部电器元件直接消毒，一旦损坏，生产商概不负责！
安全操作：电源线连接、电器元件维护需由专业电工操作，避免触电风险；管路连接需确保密封，防止渗漏导致设备故障或安全隐患。
